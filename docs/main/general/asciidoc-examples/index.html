Macros

Keyboard Macro

The keyboard macro uses the short (no target) macro syntax key(+key)*. Asciidoc displays each key exactly as entered in the document. Asciidoc separates multiple keys by a plus (e.g., Ctrl+T) or a comma (e.g., Ctrl,T). The plus is preferred.

|===
|Shortcut |Purpose

|kbd:[F11]
|Toggle fullscreen

|kbd:[Ctrl+T]
|Open a new tab

|kbd:[Ctrl+Shift+N]
|New incognito window

|kbd:[\ ]
|Used to escape characters

|kbd:[Ctrl+\]]
|Jump to keyword

|kbd:[Ctrl + +]
|Increase zoom
|===
Sidebars are used to visually separate auxiliary bits of content that supplement the main text.
Optional Title
